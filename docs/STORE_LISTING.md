## Detailed Description

Turn your X (Twitter) timeline into a hands-free podcast experience.

Instead of scrolling, just hit Play. The extension reads tweets aloud with natural-sounding AI voices while you work, code, cook, or commute. Stay informed without staring at your screen.

HOW IT WORKS
- Open x.com and click the extension icon
- Pick your topics (AI, Tech, Crypto, Business, Sports, and more)
- Hit Play and get back to work

FEATURES
- Natural AI voices — 11+ neural voices via Edge TTS, plus optional local engines
- Category filters — tap preset topics or add custom keywords to hear only what matters
- Smart skipping — automatically skips ads, promoted posts, and video-only tweets
- Keyboard shortcuts — Cmd+Shift+P to play/pause, Cmd+Shift+S to skip
- Queue preview — see the next 3 tweets coming up
- Speed control — 0.5x to 2.0x playback
- Tweet highlighting — scrolls to and highlights the tweet being read
- Quote tweet support — optionally reads quoted retweets too
- Show more expansion — automatically expands truncated tweets

PRIVACY FIRST
- No data collection. Zero analytics. No tracking.
- All processing happens on your device
- Cloud TTS (Edge TTS) sends only tweet text to Microsoft's speech service — no account info, no browsing data
- Optional local TTS engines run 100% offline
- Open source: https://github.com/ariediafauzan/x-timeline-reader

OPTIONAL: LOCAL TTS SERVER
For even more voice options, run the included Python TTS server locally. Supports Pocket TTS, Qwen3 TTS, VoxCPM2, and any HuggingFace model. The extension works fine without it using Edge TTS or your browser's built-in speech.
